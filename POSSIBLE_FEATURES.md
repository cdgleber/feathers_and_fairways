# Possible Features

Feature ideas for future development of Feathers & Fairways.

## Live & Real-Time
- **Live scoring / real-time updates** — WebSocket or SSE push for score changes during tournaments
- **Email/notification system** — Alerts for score updates, tournament start, team deadlines

## Team & League Management
- **Player trade/waiver wire system** — Mid-season golfer swaps between teams with configurable rules
- **Head-to-head matchups** — Weekly pairings between teams for additional competition
- **Draft mode for team selection** — Live pick-order draft instead of simultaneous selection
- **Multi-league support** — Multiple independent leagues sharing the same golfer pool
- **Custom scoring rules per league** — Configurable point values (e.g., eagle = +3 instead of +2)

## Tournament & Golfer Data
- **Tournament field management** — Track which golfers are in which tournament (cut tracking, withdrawals)
- **Score import from external APIs** — Pull scores from ESPN, PGA Tour, or SportsData API
- **Historical stats and analytics dashboard** — Season-over-season trends, golfer performance charts

## Social & Communication
- **Commissioner chat/messaging** — In-app league communication
- **League activity feed** — Timeline of team changes, score updates, and milestones

## Mobile & Platform
- **Mobile PWA support** — Offline access, push notifications, home screen install
- **Native mobile app** — iOS/Android via React Native or similar
